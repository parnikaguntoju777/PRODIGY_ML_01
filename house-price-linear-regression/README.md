\# House Price Prediction using Linear Regression



\## Project Overview

This project predicts house prices using a Linear Regression model based on housing features.



\## Dataset

\- train.csv

\- test.csv



\## Steps Performed

1\. Data loading and preprocessing

2\. Feature selection

3\. Train-test split

4\. Model training using Linear Regression

5\. Model evaluation

6\. Visualization of Actual vs Predicted prices



\## Output

\- actual\_vs\_predicted.png (stored in outputs folder)



\## Technologies Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Scikit-learn
